# Tosoo matlab SDK

A collection of matlab functions to work with the Tosoo device and data.

## Overview

MATLAB package for reading and processing Tosoo6 data files from the Axora device.


The `+tosoo6` package provides MATLAB functions to read and process `.tosoo6.parquet` files generated by the Axora device. The Axora is a headband device that records sleep data via 2-lead EEG and provides auditory stimulation at specific sleep phases to improve sleep quality.

## Installation

This package is designed to work within the Tosoo monorepo. Ensure you have:
1. MATLAB R2019b or later (for parquet support)
2. Add the parent directory containing the `+tosoo6` folder to your MATLAB path

To add the package to your MATLAB path:
```matlab
% Navigate to the parent directory of +tosoo6
% For example, if +tosoo6 is in /Users/jack/Documents/+tosoo6
addpath('/Users/jack/Documents');

% Or add permanently through the GUI:
% Home Tab → Environment → Set Path → Add Folder
```

Note: Do not add the `+tosoo6` folder itself to the path, but rather its parent directory (`Docuents`). MATLAB package folders (starting with '+') must be in a directory that is on the path.

## Functions

### read_metadata
Read metadata from a Tosoo6 parquet file.

```matlab
metadata = tosoo6.read_metadata('recording.tosoo6.parquet');
```

**Input:**
- `input_file` - Path to the parquet file (string or char array)

**Output:**
- `metadata` - Struct containing all Tosoo6 metadata fields

**Example:**
```matlab
metadata = tosoo6.read_metadata('sleep_recording.tosoo6.parquet');
fprintf('Device: %s\n', metadata.device_mac_address);
fprintf('Recording started: %s\n', metadata.recording_start_datetime);
fprintf('EEG sampling rate: %d Hz\n', metadata.eeg_sampling_frequency_hz);
```

### read_data
Read complete data with separate views for EEG and actigraphy.

```matlab
[full_data, eeg_data, actigraphy_data] = tosoo6.read_data('recording.tosoo6.parquet');
```

**Input:**
- `input_file` - Path to the parquet file

**Output:**
- `full_data` - Complete dataset as a MATLAB table
- `eeg_data` - Table containing only EEG samples and relevant columns
- `actigraphy_data` - Table containing only actigraphy samples and relevant columns
