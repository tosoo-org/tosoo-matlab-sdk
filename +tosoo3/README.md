# +tosoo3 MATLAB Package

MATLAB package for reading and processing Tosoo3 data files from the Axora device.

## Overview

The `+tosoo3` package provides MATLAB functions to read and process `.tosoo3.parquet` files generated by the Axora device. The Axora is a headband device that records sleep data via 2-lead EEG and provides auditory stimulation at specific sleep phases to improve sleep quality.

## Installation

This package is designed to work within the Tosoo monorepo. Ensure you have:
1. MATLAB R2019b or later (for parquet support)
2. Add the parent directory containing the `+tosoo3` folder to your MATLAB path

To add the package to your MATLAB path:
```matlab
% Navigate to the parent directory of +tosoo3
% For example, if +tosoo3 is in /Users/jack/Documents/+tosoo3
addpath('/Users/jack/Documents');

% Or add permanently through the GUI:
% Home Tab → Environment → Set Path → Add Folder
```

Note: Do not add the `+tosoo3` folder itself to the path, but rather its parent directory (`Docuents`). MATLAB package folders (starting with '+') must be in a directory that is on the path.

## Functions

### read_metadata
Read metadata from a Tosoo3 parquet file.

```matlab
metadata = tosoo3.read_metadata('recording.tosoo3.parquet');
```

**Input:**
- `input_file` - Path to the parquet file (string or char array)

**Output:**
- `metadata` - Struct containing all Tosoo3 metadata fields

**Example:**
```matlab
metadata = tosoo3.read_metadata('sleep_recording.tosoo3.parquet');
fprintf('Device: %s\n', metadata.device_mac_address);
fprintf('Recording started: %s\n', metadata.recording_start_datetime);
fprintf('EEG sampling rate: %d Hz\n', metadata.eeg_sampling_frequency_hz);
```

### read_data
Read complete data with separate views for EEG and actigraphy.

```matlab
[full_data, eeg_data, actigraphy_data] = tosoo3.read_data('recording.tosoo3.parquet');
```

**Input:**
- `input_file` - Path to the parquet file

**Output:**
- `full_data` - Complete dataset as a MATLAB table
- `eeg_data` - Table containing only EEG samples and relevant columns
- `actigraphy_data` - Table containing only actigraphy samples and relevant columns
